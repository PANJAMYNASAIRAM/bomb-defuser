##Project Description

This project is a Bomb Defuse Web App built using JavaScript. It simulates a countdown timer where the user must type the correct keyword ("defuse") before the timer reaches zero to stop the bomb.

The application demonstrates real-time countdown functionality, user input handling, and event-driven programming using JavaScript.

##Features

⏳ Countdown timer (10 seconds) 

💣 “Boom!!” message when time runs out

⌨️ User can defuse bomb by typing a keyword

🛑 Stops timer immediately on correct input

⚡ Real-time UI updates

🎮 Simple interactive game logic

##Technologies Used

HTML5 – Structure (input field & timer display) 

CSS3 – Styling (optional UI design) 

JavaScript (ES6) – Timer logic, DOM manipulation, event handling

##How It Works
Timer starts from 10 seconds using setInterval()
Countdown decreases every second
If timer reaches 0:
Displays "Boom!!"
User types "defuse" in the input field
On key event:
Input is checked
If correct:
Timer stops
Success message "you did it!!" is displayed
📷 Output
Countdown like:
10 → 9 → 8 ...
If failed:
Boom!! 💥
If successful:
you did it!! ✅
